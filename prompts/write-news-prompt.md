# Промпт для написания статей про сервера

## Контекст

Дайджест новостей про серверное оборудование (bare metal).
Формат — краткий обзор одной новости.
Аудитория: технические специалисты, которые хотят быть
в курсе новостей про сервера без маркетингового шума.

## Задача

Напиши обзор новости о выходе нового сервера Supermicro ARS‑511GD‑NB‑LCC.
Краткий пересказ: что это такое, чем отличается
от других моделей, какая польза этого сервера для бизнеса, насколько сложная его интеграция в существующую ИТ-архитектуру компаний?

## Формат

- 300 слов (не больше).
- Заголовок: до 80 символов, конкретный.
- 2 подзаголовка.
- Markdown, без HTML.
- Ссылка на источники в конце.

## Ограничения

- Без маркетинговых фраз («революционный», «прорывной»,
«меняет правила игры»).
- Только факты — без оценочных суждений.
- Не добавляй выводы «это изменит индустрию».
- Не пиши от первого лица.
